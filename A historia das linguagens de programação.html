A história da programação

A linguagem de programação é de fato a unidade fundamental do mundo atual da tecnologia. É considerado como o conjunto de comandos e instruções que damos às máquinas para realizar uma determinada tarefa. Por exemplo, se você der algum conjunto de instruções para adicionar dois números, a máquina fará isso por você e lhe dará a resposta correta de acordo. Mas você sabia que as linguagens de programação têm uma longa e rica história de evolução? E com uma preocupação semelhante, aqui neste artigo, daremos uma olhada na evolução das Linguagens de Programação ao longo do período.

No mundo da informática, temos cerca de 500+ linguagens de programação com sintaxe e recursos próprios. E se você digitar quem é o pai do computador, o mecanismo de busca mostrará o resultado para Charles Babbage, mas o pai do computador não escreveu o primeiro código. Foi Ada Lovelace quem escreveu a primeira linguagem de programação de computador e o ano era 1883.

1883

Nos primeiros dias, Charles Babbage havia feito o dispositivo, mas ele estava confuso sobre como dar instruções à máquina, e então Ada Lovelace escreveu as instruções para o mecanismo analítico. O dispositivo foi feito por Charles Babbage e o código foi escrito por Ada Lovelace para computar o número de Bernoulli. Primeira vez na história que a capacidade dos dispositivos de computador foi julgada.

1949: Linguagem Assembly

É um tipo de linguagem de baixo nível. Consiste principalmente em instruções (tipos de símbolos) que somente as máquinas podem entender. Nos dias de hoje, também a linguagem assembly é usada em programas de tempo real, como sistemas de simulação de navegação em vôo e equipamentos médicos, por exemplo - sistemas Fly-by-wire (FBW). Ele também é usado para criar vírus de computador.

1952: Autocode

Desenvolvido por Alick Glennie. A primeira linguagem de programação de computador compilada. COBOL e FORTRAN são as linguagens conhecidas como Autocode.

1957: FORTRAN

Os desenvolvedores são John Backus e IBM. Ele foi projetado para computação numérica e computação científica. O software para as sondas da NASA voyager-1 (sonda espacial) e voyager-2 (sonda espacial) foi originalmente escrito em FORTRAN 5.

1958: ALGOL

ALGOL significa ALGO rithmic L anguage. A fase inicial das linguagens de programação mais populares de C, C++ e JAVA. Também foi a primeira linguagem a implementar a função aninhada e tem uma sintaxe simples que FORTRAN. A primeira linguagem de programação a ter um bloco de código como “começar” que indica que seu programa foi iniciado e “fim” significa que você terminou seu código.

1959: COBOL

Ela representa CO MMON B usiness- O riented L anguage. Em 1997, 80% dos negócios mundiais funcionavam com Cobol. O serviço de receita interna dos EUA embaralhou seu caminho para o FMI baseado em COBOL (arquivo mestre individual) a fim de pagar as dezenas de milhões de pagamentos exigidos pela ajuda, alívio e segurança econômica do coronavírus.

1964: BASIC

Significa código de instrução simbólica para todos os fins para iniciantes. Em 1991, a Microsoft lançou o Visual Basic, uma versão atualizada do Basic A primeira versão do Basic para microcomputador foi co-escrita por Bill Gates, Paul Allen e Monte Davidoff para sua empresa recém-formada, a Microsoft.

1972: C

É uma linguagem de programação procedural de propósito geral e a linguagem de programação mais popular até agora. Todo o código que foi escrito anteriormente em linguagem assembly é substituído pela linguagem C, como sistema operacional, kernel e muitos outros aplicativos. Ele pode ser usado na implementação de um sistema operacional, sistema integrado e também no site da Web usando a Common Gateway Interface (CGI). C é a mãe de quase todas as linguagens de programação de nível superior, como C#, D, Go, Java, JavaScript, Limbo, LPC, Perl, PHP, Python e o shell C do Unix.
